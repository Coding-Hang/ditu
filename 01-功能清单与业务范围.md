# 滴兔智能体功能清单与业务范围

## 1. 清单定位

本文档是滴兔智能体工程的功能基线。后续技术架构、数据库、接口、工程实施、验收和 AI 编程任务全部以本文档的功能编号为准。

执行规则：

- 新增功能先进入本清单，再调整架构、接口、数据库、页面和验收。
- 删除功能先从本清单标记移除，再同步调整其他文档。
- 改动功能时保留原功能编号，更新功能说明和验收点。
- 后续任务描述必须引用功能编号。

## 2. 功能编号规则

| 前缀 | 范围 |
|---|---|
| `AUTH` | 登录、当前用户、Token |
| `USER` | 用户端套餐、个人信息 |
| `CHAT` | 多轮会话、聊天记录、SSE |
| `CS` | 专属客服展示 |
| `TICKET` | 用户端工单 |
| `ADMIN-USER` | 管理端用户管理 |
| `ADMIN-QUOTA` | 管理端次数管理 |
| `ADMIN-PLAN` | 管理端套餐管理 |
| `ADMIN-MODEL` | 管理端用户大模型链接管理 |
| `ADMIN-TICKET` | 管理端工单管理 |
| `RAG` | 知识库、文档、检索 |
| `AGENT` | AgentScope、Tool、Skill、Memory、事件 |
| `DEV` | 代码规范、注释、可维护性 |
| `SEC` | 安全、权限、审计 |
| `OPS` | 部署、观测、测试 |

## 11. 交付边界

完整交付范围覆盖本清单全部功能。优先级仅用于实施排序和风险控制，不用于裁剪交付范围。

核心功能包含：

- 登录和权限。
- 用户套餐和次数。
- 多轮会话。
- SSE 流式输出。
- 聊天记录。
- 专属客服列表。
- 用户工单。
- 管理端用户管理。
- 管理端次数控制。
- 管理端用户大模型链接配置。
- 管理端工单处理。
- 通用 RAG。
- 用户专属 RAG。
- AgentScope Java 接入。
- 用户级模型链接运行时解析。
- 代码注释。
- 审计日志。
- 后端集成测试和前端构建。
